# Paula's Choice Consumer Insights — Xiaohongshu NLP Analysis

Collected and analysed 118 Xiaohongshu posts and 1,914 comments 
related to Paula's Choice skincare products.

## Key steps
- Data collection via MediaCrawler
- EDA on engagement metrics and user geography  
- Rule-based vs Transformer sentiment analysis (distilbert-base-multilingual)
- Chinese RoBERTa tokenisation validation

## Tools
Python, pandas, matplotlib, transformers, MediaCrawler
